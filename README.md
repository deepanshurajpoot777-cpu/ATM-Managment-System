# ATM-Managment-System
A Simple console based ATM Managment System developed in python. This project simulates basic Atm operation such as pin authentication,balance inquiry,cash deposit,cash withdrawal,and transaction history.it is designed to strengthen core python programming concepts

ATM Management System

Description

A simple console-based ATM Management System developed using Python. This project simulates the basic functionality of an ATM and helps beginners understand Python programming through a real-world application.

Features

- PIN Authentication
- Check Balance
- Deposit Money
- Withdraw Money
- View Transaction History
- Transaction Counter

Technologies Used

- Python 3

Concepts Used

- Variables
- Conditional Statements
- Loops
- Lists
- String Formatting
- User Input Handling

About the Project

This project is designed to practice Python fundamentals by implementing an ATM system. It allows users to log in using a PIN, perform basic banking operations, and maintain a transaction history during program execution. The project demonstrates the use of loops, conditions, lists, and user interaction in a simple command-line application.
